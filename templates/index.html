📱

即時更新

📌 加入 Google Calendar

1
複製 Calendar ID
school-calendar@group.calendar.google.com
2
在 Google Calendar 中新增

其他日曆 → 從 URL 新增日曆 → 貼上上方 ID

🔗 開啟 Google Calendar

🍎 加入 Apple Calendar / iCal

1
下載 ICS 檔案

直接訂閱此 ICS URL，自動同步到你的行事曆

https://school-calendar.vercel.app/calendar/school.ics

📱 其他行事曆應用

  • 📧
    Microsoft Outlook

    訂閱 ICS URL 即可同步

  • 📄
    通用 ICS 格式

    支援所有標準行事曆應用

💬 LINE 通知訂閱

📱

加入 LINE@ 好友，接收即時放假通知

🔌 API 端點

  • GET /api/calendar

    取得所有行事曆事件 (JSON)

  • GET /calendar/school.ics

    取得 ICS 格式行事曆

  • GET /api/holidays?year=2025

    取得特定年份假期
